Inheritance vs Composition

11 September 2015

What is the difference between Inheritance and Composition in Object-oriented programming? Why might using Composition be a better way to go?

Inheritance

Inheritance is used when one class wishes to get all its features from a parent class. Syntactically, this is achieved using "<". In Ruby, each class can only inherit once from another class. The example below shows a Cat class and the breed, Persian class inheriting form the cat class and getting it's initialization and make_noise methods. While the Persian class has inherited the Cat class, it can also make additional changes such as adding colour when it gets instantiated in addition to having four legs. 

class Cat
  def initialize
    @legs = 4
  end

  def make_noise
    "Meow!"
  end
end

class Persian < Cat
 def initialize(color)
   @color = black
   super()
end

cat = Cat.new
persian = Persian.new

cat.make_noise
=> "Meow!"
persian.make_noise
=> "Meow!"

The downside of using inheritance is that it violates the encapsulation principle. You are essentially bonding one class to another so it is dependent on the parent class and it's behaviours. These behaviours may change in the future and may not be relevant to the class that is inheriting from it. It can also be hard to test classes that reuse code from the parent class.

Composition

While inheritance can be useful, you can achieve the same thing by using other classes and modules. They package of methods that can be used by other classes. The main difference between classes and modules is that modules can't be instantiated but can be included easily in other classes to be used. Below is an example of using classes for the battleships game. The player is able to create a board and access board's method receive_ship. 

 require_relative 'board'

 class Board
   attr_reader :ship_array

   def initialize
     @ship_array = []
    end

   def receive_ship(ship)
     ship_array << ship
   end
 end


 class Player
   attr_reader :board

   def initialize
     @board = Board.new
   end

   def place(ship)
     board.receive_ship(ship)
   end
 end
